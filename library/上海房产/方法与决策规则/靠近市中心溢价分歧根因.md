# 靠近市中心溢价分歧根因

## 当前判断

"靠近市中心溢价可持续性"的判断分歧，根因 = **方法论假设分歧 + 事实未决**双重来源。4 套方法论 3 轮后信心收敛到 70–72% 窄区间，对"老破小 300 万段：短期筑底、长期封顶"趋同。

## 关键机制 / 分歧结构

- **供给硬底派**（城市更新 310、资产老谢）：内环供给硬底支撑溢价，下行有界。
- **反馈环/叙事回归派**（投资毛毛、生活容器六娃）：溢价由叙事与杠杆反馈支撑，可回撤。
- 两派对同一溢价给出不同可持续性，源于对"供给 vs 叙事/杠杆"哪一个是主导因子的假设差异。

## 趋同点

- 老破小 300 万段：**短期筑底、长期封顶**。
  - 封顶三因：房龄红线（[[上海公积金管委会2023-10-房龄贷款红线|房龄贷款红线]]）+ 人口流失（[[上海市统计局2024统计公报-外来常住|外来常住 −23.79 万]]）+ 动迁期权退场（[[旧改收官与动迁期权退场]]）。

## 可证伪信号（任一翻转即松动判断）

- 居民去杠杆斜率是否企稳（[[人行上海总部-上海个人住房贷款增速|房贷增速转正]]）。
- 外来常住人口是否转正。
- 老破小房龄越过银行红线的存量占比（无公开口径）。

## 未解决问题

- 租金口径冲突（中指 −0.3% vs 房屋租赁行业协会 −8.2%）→ 影响"伪信号/价值底"判断强度。详见 [[租售比与租金信号]]。
- 上海市单独杠杆率序列、外来人口中期趋势待补。
