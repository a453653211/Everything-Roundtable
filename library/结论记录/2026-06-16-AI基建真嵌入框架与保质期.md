# AI 基建细分环节真嵌入框架（除"光"之外能入局的环节）

议题：AI 基建 + 能源转型下，除"光"外中国真正能入局的细分环节。4 席（钞能力毛毛 / 淘沙博士 / 姜汁汽水 / 王晓）3 轮。

## 结论

- 核心产出不是"标的名单"，而是"真嵌入"的三种范式 + "保质期"第五维 + 双向收敛。详见 [[AI基建真嵌入框架与保质期]]。
- 三范式：A 反向卡脖子资源型（稀土/钨，抗脱钩★★★★★）/ B 内需基建锁定型（核电泵壳等，保质期最长）/ C 顺向科技链型（光芯片嵌英伟达链，保质期最短、代际绑定）。
- 老生常谈四环节（铜缆/PCB/液冷/电源）不构成与"光"同量级的真嵌入：铜缆＝限时版半个光（保质期约 3–5 年），其余可剥离、友岸可替。
- 判断"真嵌入"用 [[份额与定价权判据]]；引用资源价格用 [[境内外价格口径分离]]。

## 成立条件

- 议题限于"中国能入局的硬科技/资源/基建细分环节"，时间锚在 2026 年中。
- 三范式保质期为定性排序，未量化年限。

## 关键依据

- [[美国防部1260H清单2026-06-08]]：1260H 口径仅中际旭创；亨通/中天属 BIS 实体清单（不同清单）。
- [[商务部出口管制公告稀土钨系2025]] + [[USGS-MCS2025-钨.pdf]] + [[厦门钨业2025年报]]：A 范式资源底座与续命方向。
- [[中国核能发展报告2025与应流股份]]：B 范式样板（核电 102 台、应流主泵泵壳唯一国产化）。
- [[Lumentum-UHP-CW激光器CPO]]：C 范式代差与产能锁定。
- [[天岳先进2025-SiC份额第一却预亏]]：份额≠定价权反例。

## 分歧与未决问题

- 替代时间债梯度（结论 D）为分析推演非公开工期数据，仅作排序用。
- 应流毛利率"约 32.8%"、光伏细钨丝"87.2%"、金力/中科三环毛利口径，均待二次核验，不写入为定论。
- 各范式保质期"最长/最短"为定性判断，缺量化区间。

## 可证伪信号

- 美/欧本土与友岸稀土/钨产能（MP、Lynas 等）投产进度 → 影响 A 范式保质期。
- 光芯片代差是否收敛（源杰 300mW 量产时点）→ 影响 C 范式保质期。
- 老生常谈四环节中是否有环节出现不可替代的认证/工艺壁垒 → 推翻 F 结论。
