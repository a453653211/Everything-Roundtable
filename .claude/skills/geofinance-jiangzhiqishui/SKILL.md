# 姜汁汽水 · 地缘货币分析法

把任何热点新闻还原为**各行为体在结构性约束下的理性选择**，再嵌入**可检验的时间预测**。
分析顺序永远是：**机制 → 约束 → 情景 → 触发 → 验证**，而不是从结论倒推。

## 一、四条公理（一切分析的地基）

1. **货币即债务/信用，不是商品。** 美元 = 美债 = 未来税收的折现期货；纸币本质是"存款凭证/欠条"，靠"借新还旧"运转。理解任何金融现象，先问"这笔钱是怎么发出来的、谁在接盘"。
2. **真实财富 = 商品与生产能力。** 印钱、发债、虚拟资产上涨都不创造财富；黄金/比特币只是信用体系的"温度计"，涨的不是资产，是货币对商品贬值。
3. **美国霸权处于结构性收缩，三条腿不可兼得。** 科技、美元、军事三线同时维持霸权不可持续；注意力倾向一线，另两线必出破口。根因是霸权衰落，不是"谁做错了什么"。
4. **万物关联，先机制后结论。** 美债、美元、股市、地缘、人民币、大宗必须放进同一框架；孤立看单一市场必错。

## 二、核心方法（已整合去重，按使用顺序）

### 1. 第一性机制还原（历史溯源法）
任何短期乱象，先溯源到底层机制再解释。习惯从历史原型讲起：元朝交子/票号超发挤兑 → 布雷顿森林 → 脱钩 → 美联储发债。**历史类比是"比机制，不比走势"**——找相同的结构（如苏联因石油-粮食-美元结算崩溃、越战印钱的财政-货币链），而非简单套用 K 线。

### 2. 资产负债表穿透 + 流动性三池
一切主体（美联储/银行/财政部/个人/稳定币机构）都用借贷记账还原其行为：先画表，再问"借什么、贷什么、钱从哪个池子流到哪个池子"。
- **流动性三池**：银行准备金 / 隔夜逆回购(ON RRP) / 财政部账户(TGA)。缩表抽干顺序决定危机形态；TGA 存钱 = 从银行体系抽水。
- **扩表必问**：买**短债还是长债**、**高息接不接**、从**谁手里买**（买谁的资产=救谁）。

### 3. 势力范围 + 三线联动（地缘地图法）
- 美国有内外两套地图：国内产业带 + 海外收割区（共和党↔中东，民主党↔俄乌/跨国资本）；"谁当韭菜"轮换，不针对特定国家的道德过错。
- **新三线**：欧洲/中东/亚太不是独立战场，是同时削弱美国科技、美元、军事三条腿且彼此联动。**分析任一热点，先问"另外两线在发生什么"。**
- **画线串联**：把离散事件（刺杀、基地、演习、访问）挂到地图的封锁线/包围圈上，"串起来就清楚"。

### 4. 行动者理性选择（目标 + 期望值）
- 区分每个行为体的**短期目标**（选举、停火、金融战）与**长期目标**（定价权、海权、统一），允许牺牲短期换长期。
- **收益期望值**：行动方在多个剧本中选**期望收益更高、执行更简单**的路径（如美亲自下场中东 > 代理人战争乌克兰）。
- **换位思考**："如果你是俄罗斯/伊朗，会等川普上台还是现在动手？"

### 5. 多剧本删枝（情景树）
对关键方向先列 **3–6 套互斥剧本**，标注概率层级与时间尺度（月/年/4–5 年），用**关键事件触发后逐一删枝**，收敛到唯一剧本。剧本是分析透镜，不是新闻线。

### 6. 表象 vs 深层
- **行动 > 表态**：政客口号是预期管理；耶伦调整发债结构、美联储 MBS 本金转投短债，比鲍威尔一句话更重要。
- **标志性事件 > headline**：用可观测、可证伪的标志替代媒体叙事（如"沙特对也门开放领空"重于"不续签美元协议"；"伊朗核设施若仍在谈判=未被摧毁"）。
- **人事 = 派系**：人事/选举先画权力结构（如伊朗总统 vs 最高领袖），再判断"亲美派上台"是否真改变阶段任务。
- **联合声明关键词**（"1967两国方案""新时代/全天候"）是立场与关系定级信号，非理想主义。

### 7. 时间轴管理（阶段 + 日历 + 触发）
- **阶段论**：战略防守 → 相持 → 进攻；重大装备/外交突破/阅兵是阶段转换的可观测里程碑。"重大军事冲突前，必先有重大外交突破。"
- **日历锚**：把转折点精确到关键月/日（FOMC、日央行、发债高峰、报税季 4 月、美国新财年 10 月、大选窗口）。
- **弱信号优先**："风起于青萍之末"——在大众媒体反应前用弱信号做早预测，早预测优于事后诸葛亮。

### 8. 路径分叉推演（不可能三角）
先识别**不能同时成立的约束**，再列分叉：
- 货币侧典型分叉：**降息 vs 扩表**（价 vs 量）、控通胀 vs 保就业、缩表 vs 保流动性。
- 不押单一结论，**列矛盾清单**比押注更重要："怎么推都会撞墙"本身就是结论。

### 9. 资产层：长期方向 vs 短期催化 + 风控纪律
- 先用长期结构（去美元化/滞胀/战略囤矿）定方向，再用短期因子（挤兑、关税、Fed、期权伽马、和谈新闻）做加减仓。**短期严重偏离长期 = 泡沫信号。**
- **涨跌同源**：什么力量推涨，衰弱后同一逻辑推跌。
- **风控铁律（以黄金为压舱石范式）**：不可空仓、不可 all in、按成本分档操作、双向防守（既防过度悲观杀跌，也防过度乐观追高）；做胜率高的操作，涨跌幅"满足"后转防守。

### 10. 预测清单 + 赛博对账（闭环校验）
- 维护**已验证 / 未验证**预测清单，公开留痕、动态更新。
- **预测错了要回去找结构性原因**（如复盘"法国奥运停火"失败 → 修正"飞地被胁迫"变量），打脸即学习。
- 用官方统计数据对冲网络叙事（"赛博对账"），强调普遍情况、不以偏概全。
- 严肃跟踪与"图一乐"免责并存：结论可换，机制链条可复用。

## 三、数据使用技巧

**盯关键阈值，信机制不信单点**（演算的精确数字仅供参考）：

| 类别 | 指标 | 用法 / 阈值 |
|---|---|---|
| 联储流动性 | 银行准备金、ON RRP 余额 | 准备金 ~2.7–3万亿为警戒；逆回购见底→被迫扩表/降息 |
| 财政抽水 | TGA 余额、报税季 | 抽走准备金，乘数放大压力 |
| 利率结构 | 2Y–10Y 利差、收益率曲线形态 | 倒挂结束→历史多伴衰退；熊陡→长债熊市/YCC |
| 资金比价 | SOFR vs 10Y | SOFR 高于长债→资金撤出债市 |
| 系统红线 | 10年期美债 | 4%–5%震荡，**4.5% 为系统性风险线**（基差套利、政策底线） |
| 美债买家 | Primary/Direct/Indirect 获配比例 | 国内机构接盘↑ = 外债内债化 |
| 通胀/就业 | CPI 粘性项、非农及**修正幅度** | 大幅下修→质疑数据/找"背锅侠"用尽 |
| 汇率 | 美元指数分阶段、人民币一篮子货币指数 | YCC 后别单看 DXY，看美元对东亚货币；人民币看一篮子 |
| 贵金属 | 金铜比、纸/实比、COMEX 交割量+cover、GOFO 租赁、ETF/央行实物持仓 | 交割↑+cover↓=挤兑；价涨仓降=投机过热；单指标会失效，须组合 |
| 大宗 | 库欣库存、炼油产能、铜矿建设周期、COT 商业净空 | 供给周期 ≥ 需求叙事 |

**原则**：供给端优先（美债问题是美国财政供给问题，不是"中国不买"）；用"差分对比"（同一行为在不同战场的差异→推优先级）；承认数据局限仍坚持用最客观来源并做高/低敏感性。

| 流动性/利率市场 | 银行准备金、隔夜逆回购(ON RRP)余额、**SOFR** 与利率走廊、补充杠杆率(SLR/eSLR)、基差套利数据 |
| 黄金/白银 | **CME（芝加哥商品交易所）黄金/白银交割报告**与 5 年交割量、**GOFO（黄金租赁）spread**、现货 bid-offer 价差、ETF/央行实物持仓 |
| 铜/大宗/原油 | 金铜比、**COMEX registered/eligible**、CME 保证金调整、**COT 商业净持仓**、库欣原油库存、炼油产能 |
| 地缘/外交 | **官方联合声明与官方表态**（如"1967两国方案"、外交关系措辞）、政府网站外交等级表述、竞选金主/献金结构；明确区分官方与非官方消息源 |

## 四、一期分析的展开骨架

1. 时效钩子（今日日期 + 本期问题）
2. 机制补课（"不先讲发行机制你会听不懂"——溯源历史/原理）
3. 立框架/空间 setup（先讲思路与总原则，再画地图、三线、包围圈）
4. 分行为体动机链（美/以/俄/伊/欧/日，常先讲对手再讲己方）
5. 机制层（货币环流、关税-GDP、流动性、滞胀、门罗矛盾）
6. 情景树 + 概率 + 时间表
7. 近期新闻速扫（验证或修正上述树）
8. 对自身的启示/应对原则 + 风险声明（结论可能错、不荐具体投资品）

## 五、应用检查清单（分析任意热点时逐条自问）

- [ ] 这件事的**底层机制**是什么？能否用资产负债表/发行链/势力范围还原？
- [ ] 放进**三线/全局**看，另外两线在发生什么？
- [ ] 各**行为体的短期 vs 长期目标**分别是什么？哪个剧本期望收益最高、最易执行？
- [ ] 表象之下的**深层原因**？哪个是**标志性事件**而非媒体噪声？**行动**和**表态**是否一致？
- [ ] 列出 **3–6 套互斥剧本** + 每套的**触发条件**与**时间窗口**。
- [ ] 存在哪些**不能同时成立的约束**（不可能三角）？
- [ ] 长期方向与短期催化是否一致？若背离，是否是**泡沫/超跌**信号？
- [ ] 给出**可证伪的预测**并登记到清单；预留复盘与修正。
- [ ] 区分"对手**想做什么**"与"对手**能否做成**"。
