# 钞能力毛毛 · 投资机遇分析法

把一个领域**用思维导图结构化讲透**：先抛结论，再拆机制（是什么→为什么赚钱/亏钱），
用原始数据与国际镜鉴验证，最后落到**可参与的产业链 + 拐点窗口 + 风险边界**。

## 一、四条公理（地基）

1. **市场非有效，价格 = 理性 + 情绪。** 不存在可靠的"合理估值"与均值回归；有人参与就有"肥沃的谬误"，泡沫与崩溃内生于机制——**可以与泡沫共舞，但必须能识别拐点**。
2. **经济长周期由技术革命驱动。** 约 40–60 年一轮；旧红利出尽则下行，新革命开启则上行。"新质生产力" ≈ 本轮革命的具体清单。宏观引擎已从地产切换到科技，流动性是**结构性滴灌**而非大水漫灌。
3. **政策文件是"强度信号"，不是口号。** 要把多年报告**横向并排**比词频、顺序、数字（赤字率、M2 目标），而非读单年"积极/稳健"的形容词。
4. **只信逻辑自洽。** 传闻、标题党、单点数据都要回到供需/机制/成本能否**闭环**；溯源原始论文、扣除全部成本后再谈回报。

## 二、核心方法（已整合去重）

### 1. 反馈环模型（统一框架）
识别"价格 ↔ 预期 ↔ 基本面/情绪"是否**同向自我加强**，加强到脆弱后被小事件触发**自我削弱**。三变量同向 → 趋势延续；出现背离 → 拐点临近。**此模型统一解释**：股票繁荣-萧条、成长股"杀估值"、房价法拍螺旋、政策市碳价。用法：跟踪股价/基本趋势/主流情绪三者，确认加强则参与、削弱信号出现则退出。

### 2. 人造收藏品的"通缩供给"框架（另类投资通用）
**供给通缩 × 需求扩张 × 退出通道清晰 × 买得便宜**，四者缺一不可。关键追问：
- 接盘买家是谁？能持续 5–15 年吗？（有无接盘生态）
- 厂商的停产/重置/复刻周期？（**厂商掌握通缩/通胀开关**）
- 折扣 = 品牌犯错替你付的价；要**原价/折扣价**入手控波动。
- 扣除仓储/物流/平台费/货损/保养/保险后的**真实年化**是多少？
- **不与品牌争利**：热门款是品牌的钱袋子，必扩产补货复刻；要找"冷中带热、热中带冷"的中间地带。

### 3. 宏观流动性与信用（钱从哪来、去哪、能否进股/楼）
- **放水四去向**：好实业 / 坏实业（以贷养贷）/ 股市（快牛快熊）/ 房贷（锁通胀+财富效应）。据此判断本轮是"真技术周期"还是"续命并酝酿下一轮危机"。
- **政策三层解码**：① 流动性总量松不松？② 风险兜不兜底？③ 增长方向如何排序？（产业排序前移 = 财政注意力上升）
- **化债 = 一边减一边加**：银行/间接融资去杠杆，同时股市/股权融资加杠杆；信用置换 + REITs 消化库存。用宏观杠杆率、M2/GDP、金融增加值/GDP 三角诊断，并对标美/日。

### 4. 产业机遇链（标准推演路径）
```
顶层叙事（新质生产力/两会排序）
  → 瓶颈环节（缺货、国产化率低、政策点名）
    → 商业模式是否改善（ROE、定价权、二供格局）
      → 估值是否已透支（PE 分位、杀估值风险）
        → 技术/政策拐点是否威胁线性外推
```
- **国产替代"二供 + 缺货"密码**：海外垄断 + 大客户（云厂/NVIDIA）要供应链安全 → 扶持二供 → 叠加政策/产能 → 窗口期业绩爆发。观察政策点名、并购引才、全球份额洗牌、上游国产化率爬升。
- **新质生产力七板块地图**：把抽象政策翻译成正在发生的能源/计算/农业等革命，每块找"卖水人/配套/门槛低的 service"（如低空经济→停机坪/智能柜；柔性供应链→直播+小单快反）。

### 5. 代际/路径跃迁模板（技术-政策-商业化三角）
重大技术或物理极限会**重写产业链地位与毛利**，必须分轨判断：
- 安全/性能代际跃迁 → 政策态度翻转 → 经济性（回收周期）→ 商业化进度（国家订单 vs 资本先行）。
- **用物理/工程极限标定拐点年份**（如光模块 1.6T 可插拔极限→2028 CPO；核电四代固有安全）。
- **线性外推有边界**：明确"只能外推 1–2 年"，拐点后必须换模型、看转型生死线。

### 6. 房价：三阶段 + 日美镜鉴
- **三阶段**：① 只涨不跌（人口+货币协同）→ ② 只跌不涨去库存（乱纪元：法拍、供给收紧）→ ③ 平衡后货币驱动再涨。劳动力人口拐点 ≈ 房价拐点。
- **双轴**：长期看商品（人口/人均面积/城市化）+ 金融（M2 温和 7–8%"为王"）+ 制度；短期博弈看**成交量 + 法拍占比**（>5% 打破均衡，20–30% 深度下跌）。
- **镜鉴**：美国 6 年 vs 日本 16 年去库存；止跌后涨幅向"二线承接 + 特色城市"迁移。

### 7. 历史镜鉴原则（贯穿所有类比）
先找**相同机制**（人口拐点+货币宽松→泡沫；供过于求+货币无法单向上推→去库存），再比**政策组合差异**（美国快速收紧供给 vs 日本政策失误），最后映射**进度条**而非点对点预测年份。

## 三、数据使用技巧

| 技巧 | 做法 |
|---|---|
| 溯源原始数据 | 不信标题/传闻，翻到论文出处页核对；用第三方数据站点交叉验证（见下节"常用数据源"） |
| 扣全成本算真实年化 | 仓储、物流、平台费、货损、保养、保险全部扣除后再谈回报 |
| 国际对标标尺 | 中/美/日同指标并排：杠杆率、M2/GDP、金融增加值/GDP、去库存年限 |
| 政策数字 > 形容词 | 赤字率 3.2% vs "积极"；M2 目标数 vs "适度宽松" |
| 简化量化模型 | 投资额 vs 回本年限（如三代核电 15 年 vs 四代 7 年）、单款 ±波动率 |
| 物理/工程极限定拐点 | 良率、K 值、上游 18–24 月周期、架构极限年份 |
| 资金面验证泡沫 | 泡沫期看开户/公募私募增量资金周报，而非叙事 |

## 三续、常用数据源（按典型问题，均为字幕中明确点名/使用者）

> 仅收录材料里**确有出处**的；他常说"找原始数据、翻到第几页、交叉验证"，对未点名的只统称"数据网站/图表"。

| 典型问题 | 常用数据源（字幕原话/可考） |
|---|---|
| 另类资产能否升值（乐高/潮玩） | 引用的**学术论文**原始数据（如乐高 "investors" 2016 论文，强调"翻到第四页"看口径）；**乐高投资/数据网站**与 **BrickLink** 部件交易平台；乐高**官网退役标签/折扣**；电商平台成交最高/最低价 |
| 名表投资 | **WatchCharts** 网站及其自编 **watch index**（按年化回报、流动性、价格可预测性评级）；与标普500长期对比 |
| 读懂宏观政策方向 | **政府工作报告 / 两会《报告》/ 五年规划**原文——多年并排比词频、排序、数字（赤字率、M2 目标、社融） |
| 房价涨跌 | **法拍占比**、库存月数/去化、新房供给降幅、成交量；日/美同阶段历史数据做"镜鉴进度条" |
| 化债/经济棋局 | 中/美/日横向对标：宏观杠杆率、M2/GDP、金融增加值占 GDP |
| 产业赛道机会（核电/光模块/量子等） | 相关公司/机构**官网**案例与里程碑时间线；技术路线图与物理极限年份；行业全球份额、国产化率；资金面"开户/公募私募增量"周报 |

## 四、识别伪逻辑（反向清单）

- "只报涨幅不报成本/不报单款波动" → 用中位数、单款 ±50%~600% 打脸。
- "产量低 = 一定升值" → 反例：百达翡丽 5万只 vs 劳力士 100万只，逻辑不成立。
- "热门 = 值得囤" → 厂商必扩产/重置，与品牌争利必输。
- "贸易战一句话就能结束" → 用财政赤字缺口 + 竞选金主结构证伪政策刚性。
- "碳市场 = 自由炒碳" → 供给由国家控盘，须读罚款/抵扣限制/MSR 等规则组合拳。
- **挖信息差**：成熟赛道存在数据网站本身 = 有利可图；用海外（如欧盟碳市场 20 年）试错史预测国内节奏。

## 五、一期分析的展开骨架

1. 钩子 + **结论前置**（"老规矩先抛结论"）
2. **思维导图/全景图**（可暂停截图）
3. 机制拆解（是什么 → 为什么赚钱/亏钱）
4. 时间线复盘（N 重 Buff 如何叠加成今天）
5. 数据验证（原始论文 + 数据网站 + 国际对标）
6. 量化测算（简化模型、年化、回本）
7. 机遇 vs 风险**双栏**（各 3 条左右）
8. 推演未来（拐点/窗口 + 明确线性外推范围）
9. 落地建议（该看什么数据、什么坚决别碰）
10. 2 分钟精华总结 + 系列预告

## 六、应用检查清单（评估任意机会时逐条自问）

- [ ] 先用一句话**抛出结论**，再画**思维导图**列出全部机制环节。
- [ ] 这个逻辑**自洽吗**？数据**溯源**了吗？**扣除全部成本**后回报还成立吗？
- [ ] （另类资产）供给通缩 × 需求扩张 × 退出通道 × 买得便宜，**四项是否齐全**？是否在**与品牌争利**？
- [ ] （宏观）钱**从哪来、去哪**，会不会进股/楼？政策**三层信号**（松紧/兜底/方向）是什么？
- [ ] （产业）顶层叙事→瓶颈→商业模式→估值→拐点，**链条是否走通**？是否已**杀估值**？
- [ ] 是否存在**技术/政策拐点**重写产业链？我的**线性外推**最多能推几年？
- [ ] 历史类比：**机制是否真的相同**？政策差异在哪？我比的是**进度条**还是硬套年份？
- [ ] 拐点/窗口的**触发条件**是什么？机遇与风险**各列三条**了吗？
